## Professional patch holati
- **Talaba profili tablari:** student_notes, student_discounts, student_exam_results, crm_messages, student_tasks, gamification_transactions, crm_history jadvallari va API endpointlar qo‘shildi.
- **Guruh profili:** group profile API, group exams, homeworks, notes, tasks, history endpointlari qo‘shildi.
- **To‘lov/qarzdorlik:** payment status engine va balance recalculation mavjud, CRUD saqlanadi.
- **Student App:** demo fallback default o‘chirildi, real DATABASE_URL va sessiya talab qilinadi.
- **Demo rejim:** /api/auth/demo 410 qaytaradi; frontend mock fallback faqat ?demo=1 yoki eduka_allow_demo=1 bo‘lsa ishlaydi.
- **Import/export/report:** oldingi action patch saqlandi; backend bilan ishlamagan joylarda noaniq demo ko‘rsatish o‘rniga xato ko‘rsatadi.
