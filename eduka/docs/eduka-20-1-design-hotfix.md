# Eduka 20.1 — Design Hotfix

## Maqsad
Eduka 20.0 dan keyin qolgan dizayn muammolarini yumshatish: loader, Super Admin formasi, card/table/filter/button ko'rinishi, Student App vizuali va eski CSS/JS cache muammosini kamaytirish.

## O'zgarishlar
- `app.html` va `student-app.html` asset versiyalari `20.1.0` ga ko'tarildi.
- Static fayllar uchun `X-Eduka-Version` `20.1.0` ga yangilandi.
- Super Admin yangi markaz yaratish formasi 4 bosqichli professional ko'rinishga keltirildi.
- “Boshlang'ich ma'lumot bilan yaratish” checkboxiga aniq izoh qo'shildi.
- Card, table, filter, button, modal va formalar uchun premium SaaS CSS polish qo'shildi.
- Student App uchun card/button/input dizayni yangilandi.
