# Eduka 31.1.0 — Premium CRM Dashboard & Design System

## Maqsad
Admin dashboard va umumiy CRM ko‘rinishini premium SaaS darajasiga olib chiqish.

## Qilingan ishlar
- Dashboard 4 blokli yangi konseptga o‘tkazildi: Hero, KPI cards, Today Operations, Action Center.
- Sidebar kategoriyalarga ajratildi: Asosiy, Moliya, Sotuv, Platforma.
- Topbar minimal qilindi: Search, markaz nomi, notification, language, profile.
- KPI cardlar premium glass style bilan qayta ishlangan.
- Eski dashboard bloklari Pro dashboard route’da yashiriladi.
- Bitta premium design token tizimi qo‘shildi.
- Mobile responsive dashboard polish qilindi.
