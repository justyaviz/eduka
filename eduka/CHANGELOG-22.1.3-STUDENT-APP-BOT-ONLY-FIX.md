# Eduka 22.1.3 — Student App Bot-Only Login & Crash Fix

## Fixes
- Fixed Railway crash caused by `schedule_enabled` missing on old Railway Postgres databases.
- Added migration `011_student_app_2213_safe_autologin.sql`.
- Added hotfix SQL `backend/hotfix-22.1.3-student-app-safe.sql`.
