# Eduka CRM v20.2.0 — Student & Group Flow Upgrade

## Nimalar yangilandi
- Student profile sahifasi LC-UP uslubiga yaqinroq qilib qayta yig'ildi.
- Group profile sahifasi attendance / grading / homework / discounts / exams / history / note tablari bilan yangilandi.
- Student drawer (Yangi talaba qo'shish) to'liq qayta kompozitsiya qilindi.
- Group drawer (Yangi guruh qo'shish) kun, vaqt, o'qituvchi, maosh va format maydonlari bilan kengaytirildi.
- Tanga/Kristal hisobi, to'lovlar, tarix va note panellari strukturali ko'rinishga o'tkazildi.
- Profile tab switching real ishlaydigan holatga o'tkazildi.
- Drawer responsive width va sticky footer yaxshilandi.
- Asset/API version 20.2.0 ga ko'tarildi.

## Fokus qilingan joylar
- Talabalar ro'yxati -> talaba detail -> to'lovlar/tarix/coin-crystal
- Guruhlar ro'yxati -> guruh detail -> davomat/baholash/mashqlar
- Yangi talaba qo'shish modal/drawer
- Yangi guruh qo'shish modal/drawer

## Keyingi tavsiya qilinadigan bosqich
- O'qituvchilar detail flow ni shu uslubga olib chiqish
- Lead / Moliya / Hisobotlar sahifalarida vizual consistency hotfix
